```js
i18n: {
  locales: ['en', 'es'],
  defaultLocale: 'en',
  localeDetection: true,
}
```

### Librería recomendada

**`next-intl`** — integración nativa con Next.js App Router, soporte TypeScript, sin overhead de bundle.

### Estructura de archivos de traducción

```
messages/
  en.json   ← todos los strings EN
  es.json   ← todos los strings ES
```

### hreflang tags *(en `<head>` de cada página)*

```html
<link rel="alternate" hrefLang="en"       href="https://[dominio.com]/" />
<link rel="alternate" hrefLang="es"       href="https://[dominio.com]/es/" />
<link rel="alternate" hrefLang="x-default" href="https://[dominio.com]/" />
```

### Selector de idioma en Nav

| Estado | ES | EN |
|--------|----|----|
| Label botón activo | ES | EN |
| Tooltip hover | Cambiar a English | Cambiar a Español |
